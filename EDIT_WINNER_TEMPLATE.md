### Step 2: Find and Replace

Use Word's Find & Replace (Ctrl+H) to add placeholders:

#### Replace Reference Number
- **Find**: Your current reference number (e.g., "033/2018")
- **Replace with**: `{refNumber}`

#### Replace Date
- **Find**: Your current date (e.g., "25/08/2017")
- **Replace with**: `{date}`

#### Replace Winner Name
- **Find**: The winner's name in your document
- **Replace with**: `{winnerName}`

#### Replace Tender Number
- **Find**: Tender number in the body text
- **Replace with**: `{tenderNumber}`

#### Replace Group Code
- **Find**: Group code (e.g., "CODE-10")
- **Replace with**: `{groupCode}`

#### Replace Item Description
- **Find**: Item description (e.g., "የሞባይል ቀፎ")
- **Replace with**: `{itemDescription}`

#### Replace Prices
- **Find**: The winning price amount
- **Replace with**: `{winnerPrice}`

- **Find**: 70% amount
- **Replace with**: `{calc70}`

- **Find**: 30% amount
- **Replace with**: `{calc30}`

- **Find**: VAT amount
- **Replace with**: `{vat}`

- **Find**: Total amount
- **Replace with**: `{totalWithVAT}`
